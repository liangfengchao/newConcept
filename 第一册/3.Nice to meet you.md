### 语法

#### 主语 + 系动词（Be 动词） + 表语

|          |     |     |             |
| -------- | --- | --- | ----------- |
| 人称代词 | I   | you | he、she、it |
| Be 动词  | am  | are | is          |

Sophie is a new student.She is French
Sophie : 主语
is : Be 动词
a new student : 表语
表语： 主语的身份、性质、品行、特征和状态的。
不定冠词 a / an
